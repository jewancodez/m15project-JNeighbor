Instructions

Create a new Repository

  • After creating an account or signing in, navigate to repositories
  • Click "New"
  • Configure parameters like name and public/private status.
  • Click "Create repository"

Git Initialization Code in Terminal (BASH)

  • git init
  • git add .
  • git commit -m
  • git branch -M main
  • git remote add origin "your_repo_url"
  • git push -u

Launch Site

  • Check for uploaded files in repository
  • Navigate to "Settings"
  • In the left nav pane, click "Pages"
  • A link to your site will be live at a displayed URL. Click it or "Visit Site" to open it.
